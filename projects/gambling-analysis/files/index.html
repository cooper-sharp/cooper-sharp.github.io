Analysis 1: Age Distribution Across Populations

Using GROUP BY to compare age ranges across three study populations

SQL QUERY
SELECT CASE WHEN source = 'ad_data' THEN 'Adolescent/Young Adult' WHEN source = 'dvs_data' THEN 'Older Adult' WHEN source = 'g25_data' THEN 'General Population' END as Population, demo_yrs as Age, COUNT(*) as Count FROM gambling_data WHERE demo_yrs IS NOT NULL GROUP BY source, demo_yrs ORDER BY Population, Age

💡 Key Insights

  • Three distinct age ranges: adolescents (18-29), older adults (56-77), and general population (18-70)

  • GROUP BY enables aggregation across multiple dimensions

Analysis 2: Gambling Prevalence by Demographics

Conditional aggregation using CASE within SUM functions

SQL QUERY
SELECT CASE WHEN source = 'ad_data' THEN 'Adolescent' WHEN source = 'dvs_data' THEN 'Older Adult' WHEN source = 'g25_data' THEN 'General Pop' END as Age_Group, CASE WHEN demo_gender = '1' THEN 'Male' WHEN demo_gender = '2' THEN 'Female' ELSE 'Other' END as Gender, COUNT(*) as Total_Participants, SUM(CASE WHEN prob_gam = 'yes' THEN 1 ELSE 0 END) as With_Gambling_Issues, ROUND(100.0 * SUM(CASE WHEN prob_gam = 'yes' THEN 1 ELSE 0 END) / COUNT(*), 2) as Prevalence_Rate FROM gambling_data WHERE demo_gender IS NOT NULL AND prob_gam IS NOT NULL GROUP BY source, demo_gender HAVING COUNT(*) >= 10 -- Filter small groups ORDER BY Age_Group, Prevalence_Rate DESC
Loading visualization...

💡 Key Insights

  • Demonstrates conditional aggregation: CASE inside SUM for counting specific conditions
  • HAVING clause filters out small sample sizes for statistical reliability
  • Shows gender and age differences in gambling problem prevalence

Analysis 3: Mental Health by Gambling Status

Using Common Table Expressions (CTEs) for readable, complex queries

SQL QUERY
WITH mental_health_data AS ( SELECT CASE WHEN prob_gam = 'yes' THEN 'Has Gambling Issues' ELSE 'No Gambling Issues' END as Gambling_Status, CAST(susd_depression AS FLOAT) as Depression, CAST(susd_mania AS FLOAT) as Mania, CAST(bisbas_bis AS FLOAT) as Behavioral_Inhibition FROM gambling_data WHERE susd_depression IS NOT NULL AND susd_mania IS NOT NULL AND source = 'ad_data' ) SELECT Gambling_Status, COUNT(*) as N, ROUND(AVG(Depression), 2) as Avg_Depression, ROUND(AVG(Mania), 2) as Avg_Mania, ROUND(AVG(Behavioral_Inhibition), 2) as Avg_BIS FROM mental_health_data GROUP BY Gambling_Status
Loading visualization...

💡 Key Insights

  • CTEs improve query readability by breaking complex logic into named subqueries
  • Shows correlation between gambling issues and elevated mental health symptoms
  • CAST function ensures numeric operations on text-stored numbers

Analysis 4: Geographic Distribution

State-level aggregation with statistical filtering using HAVING

SQL QUERY
SELECT demo_state as State, COUNT(*) as Total_Participants, SUM(CASE WHEN prob_gam = 'yes' THEN 1 ELSE 0 END) as With_Issues, ROUND(100.0 * SUM(CASE WHEN prob_gam = 'yes' THEN 1 ELSE 0 END) / COUNT(*), 2) as Prevalence_Rate FROM gambling_data WHERE demo_state IS NOT NULL AND prob_gam IS NOT NULL GROUP BY demo_state HAVING COUNT(*) >= 10 -- Minimum sample size ORDER BY Prevalence_Rate DESC LIMIT 20
Loading visualization...

💡 Key Insights

  • HAVING clause filters aggregated results (vs WHERE for row-level filtering)
  • Geographic variation suggests environmental or policy influences
  • Top 20 states show significant range in prevalence rates

Analysis 5: Risk Factor Analysis by Age

Complex CTE with conditional aggregations across age categories

SQL QUERY
WITH risk_data AS ( SELECT prob_gam, CAST(bisbas_reward_res AS FLOAT) as Reward_Sensitivity, CASE WHEN demo_yrs < 25 THEN '18-24' WHEN demo_yrs BETWEEN 25 AND 34 THEN '25-34' WHEN demo_yrs BETWEEN 35 AND 54 THEN '35-54' WHEN demo_yrs >= 55 THEN '55+' END as Age_Category FROM gambling_data WHERE prob_gam IN ('yes', 'no') AND bisbas_reward_res IS NOT NULL ) SELECT Age_Category, ROUND(AVG(CASE WHEN prob_gam = 'yes' THEN Reward_Sensitivity END), 2) as Gambler_Reward, ROUND(AVG(CASE WHEN prob_gam = 'no' THEN Reward_Sensitivity END), 2) as NonGambler_Reward, COUNT(*) as N FROM risk_data WHERE Age_Category IS NOT NULL GROUP BY Age_Category
Loading visualization...

💡 Key Insights

  • Nested CASE statements create age bins and conditional averages
  • Shows reward sensitivity differences between gamblers and non-gamblers across age groups
  • Demonstrates advanced SQL: CTE + conditional aggregation + derived categories

🔧 Project Methodology

1

Data Import & Database Creation

Single CSV file (2,560 rows × 104 columns) imported into SQLite. Created indexes on key columns (source, prob_gam, demo_gender, demo_state, demo_yrs) for optimal query performance.

2

SQL Query Design

Wrote 5 progressively complex SQL queries demonstrating: CASE statements, conditional aggregation (CASE within SUM/AVG), Common Table Expressions (CTEs), GROUP BY with HAVING, and statistical filtering.

3

Data Analysis with Python

Used Pandas to execute SQL queries and process results. Handled NULL values, performed type casting, and cleaned data for visualization.

4

Interactive Visualization

Created 5 interactive Plotly visualizations: overlapping histograms, grouped bar charts, multi-panel comparisons, horizontal bar charts, and comparative analyses. All visualizations support zoom, pan, and hover interactions.

5

Dashboard Development

Built responsive HTML5/CSS3 dashboard with syntax-highlighted SQL code blocks, embedded visualizations, and comprehensive documentation. Designed for both desktop and mobile viewing.

Technologies Used: Python, Pandas, SQLite, Plotly, HTML5, CSS3, JavaScript
